# Python Project

## Game Features
- Detailed gameplay mechanics
- Interactive character classes
- Multiple story paths
- In-game shop for item purchases
- Dynamic game stats tracking

## How to Run
1. Install Python 3.x.
2. Clone the repository.
3. Navigate to the project directory.
4. Run the main game file using the command: 
   ```
   python main.py
   ```

## File Descriptions
- `main.py`: The main entry point of the game.
- `characters.py`: Contains definitions for character classes.
- `shop.py`: Handles shop functionalities for item transactions.
- `story.py`: Manages different story paths and their progression.

## Character Classes
- **Warrior**: High strength, equipped with melee weapons.
- **Mage**: Uses magical abilities, medium agility, but low defense.
- **Rogue**: Fast and agile, specializes in stealth and agility.

## Companions
- **Companion A**: Provides support in battles.
- **Companion B**: Has unique skills that aid in quests.

## Story Paths
- **Path 1**: Description of the first story arc.
- **Path 2**: Description of the second story arc.

## Shop Items
- **Healing Potion**: Restores health.
- **Magic Scroll**: Provides magic enhancements.

## Game Stats
- **Health**: Represents the character's health.
- **Strength**: Affects melee damage.
- **Agility**: Influences dodging ability.

## Code Highlights
- Use of OOP concepts for character classes.
- Modular file structure for better maintainability.
